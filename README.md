# 🎲 Dice Rolling Simulator (Java)

A simple command-line based dice rolling simulator written in Java. This project allows users to roll one or more dice and displays both the numerical result and a visual ASCII representation of each dice face.

---

## 📌 Features

* Roll multiple dice at once
* Random number generation using Java’s `Random` class
* ASCII-based dice face visualization
* Displays total sum of all dice rolled
* Input validation for number of dice

---

## 🛠️ Tech Stack

* Java
* Standard Library (`java.util.Random`, `java.util.Scanner`)

---

## 🚀 How It Works

1. User enters the number of dice to roll
2. Program generates random values between 1–6
3. Each roll is displayed along with its ASCII representation
4. Total sum is calculated and printed

---

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/krisjscott/Dice-rolling.git
cd Dice-rolling
```

### 2. Compile the program

```bash
javac Main.java
```

### 3. Run the program

```bash
java Main
```
